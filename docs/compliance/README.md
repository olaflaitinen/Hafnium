## Regulatory Scope

The Hafnium platform is designed to support compliance with:

| Regulation | Jurisdiction | Applicability |
|------------|--------------|---------------|
| AML Directives (4th, 5th, 6th) | EU | Financial institutions |
| Bank Secrecy Act (BSA) | US | Financial institutions |
| GDPR | EU | Personal data processing |
| CCPA | California, US | Consumer data |
| PCI DSS | Global | Payment card data |
| SOC 2 | Global | Service organizations |

## Compliance Responsibility

The Hafnium platform provides technical capabilities to support compliance. However:

1. **Compliance is the user's responsibility**: Organizations must implement appropriate policies and procedures
2. **Platform is not pre-certified**: Independent certification required
3. **Configuration required**: Security controls must be properly configured
4. **Ongoing monitoring required**: Compliance requires continuous attention

## Audit Support

The platform provides audit support through:

- Immutable audit logs
- Access control documentation
- Configuration documentation
- Change management records
