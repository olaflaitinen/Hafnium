# Hafnium Risk Engine

## Overview
The Hafnium Risk Engine is a unified risk scoring service that aggregates signals from rules, ML models, and graph analysis to provide real-time risk assessments.

## Features
- **Real-time Scoring**: Low-latency risk evaluation.
- **Explainability**: Detailed reason codes and policy actions.
- **Integration**: Seamless integration with the Hafnium Data Platform.

## Development
This service is built with Python 3.11+ and FastAPI.

### Installation
```bash
pip install -e ".[dev]"
```

### Testing
```bash
pytest
```
